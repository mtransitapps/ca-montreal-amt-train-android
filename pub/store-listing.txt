Full description:
BETA: TEST IN PROGRESS
To leave the test: http://goo.gl/WGKrhZ

This application adds Montreal AMT train information in MT.

Once this application is installed, the MT app will display train information (schedule...).

This application only has a temporary icon and no screen: download the MT app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5m

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Agence Métropolitaine de Transport.
http://www.amt.qc.ca/developers/

This application is free and open-source:
https://github.com/mtransitapps/ca-montreal-amt-train-android

This app is not related with the Agence Métropolitaine de Transport.
